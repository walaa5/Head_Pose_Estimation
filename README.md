# Head_Pose_Estimation
Predict the three angles of head_pose using mediapipe and machine Learning.
> used mediapipe to get face mesh points.<br>
> used Machine Learning to predict the angles of the face 

# Requirements:
* mediapipe
* cv2
* Sklearn
